# AI Meeting Assistant

一个基于大模型的会议纪要结构化提取项目，用于自动提取会议主题、关键决策、待办事项和风险点，并输出标准化 JSON 结果。

## Features
- 提取会议主题
- 提取关键决策
- 提取待办事项及责任人
- 提取风险点和未解决问题
- 输出结构化 JSON

## Tech Stack
- Python
- Prompt Engineering
- JSON Schema
- RAG（规划中）
